# SIGMA Database Skill

# Objetivo

Esta skill define toda a arquitetura de persistência do SIGMA Studio.

O banco deve ser capaz de armazenar grandes volumes de telemetria industrial mantendo desempenho, integridade e facilidade de manutenção.

Todo o modelo deve ser preparado para crescimento contínuo.

Nunca modelar apenas para o estado atual do projeto.

---

# Banco Oficial

PostgreSQL

Versão mínima

16

ORM

SQLAlchemy 2

Migrações

Alembic

---

# Filosofia

O banco representa a fonte oficial da verdade.

Toda alteração permanente deve passar por ele.

Nunca utilizar o banco apenas como armazenamento.

Ele deve representar corretamente o domínio do sistema.
