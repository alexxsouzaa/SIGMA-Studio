# SIGMA Frontend Skill

# Objetivo

Esta skill é responsável por toda implementação do Frontend do SIGMA Studio.

Seu objetivo é desenvolver uma interface moderna, altamente organizada, escalável e preparada para aplicações industriais de monitoramento em tempo real.

Todo código produzido deve priorizar:

- organização
- reutilização
- desempenho
- previsibilidade
- facilidade de manutenção

---

# Papel

Esta skill atua exclusivamente na camada Frontend.

Nunca implementar lógica de negócio.

Nunca implementar acesso direto ao banco.

Nunca implementar comunicação MQTT diretamente.

Nunca implementar acesso Serial diretamente.

Todo acesso externo ocorre exclusivamente através da API do Backend.
