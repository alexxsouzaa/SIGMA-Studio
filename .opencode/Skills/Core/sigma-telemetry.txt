# SIGMA Telemetry Skill

# Objetivo

Esta skill define o padrão oficial de telemetria do ecossistema SIGMA.

Ela estabelece como todos os dados produzidos pelo firmware devem ser representados, transportados, armazenados e consumidos.

Todo protocolo de comunicação deverá respeitar este documento.

A telemetria representa a linguagem oficial do SIGMA.

---

# Filosofia

O Firmware produz dados.

A Telemetria organiza esses dados.

Os protocolos apenas transportam.

Nunca permitir que MQTT, HTTP ou WebSocket alterem o significado das informações.

# Unidade Fundamental

Toda comunicação deve utilizar uma estrutura denominada:

TelemetrySample

Ela representa uma fotografia completa do estado do dispositivo em um instante específico.

Nunca transmitir variáveis isoladas.
