# SIGMA Design System Skill

# Objetivo

Esta skill é responsável por toda a identidade visual do SIGMA Studio.

Seu objetivo não é desenhar telas bonitas.

Seu objetivo é criar uma interface consistente, previsível, reutilizável e escalável.

Todo componente criado deve parecer fazer parte do mesmo software.

Nunca criar interfaces diferentes entre si.

---

# Missão

Garantir que todo o sistema possua uma identidade única.

Todas as telas devem transmitir:

- confiabilidade
- engenharia
- tecnologia
- precisão
- organização
- simplicidade

O SIGMA Studio não é um sistema administrativo.

Ele é uma plataforma industrial.

A interface deve transmitir esta característica.

---

# Inspirações

As interfaces devem seguir referências de softwares industriais modernos.

Inspirar-se em:

- Siemens TIA Portal
- Beckhoff TwinCAT
- Ignition SCADA
- Grafana

- JetBrains IDEs
- Figma
- Linear

Apenas utilizar como referência de qualidade.
